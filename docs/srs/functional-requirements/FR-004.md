# FR-004: Emergency-Stop Button

## Requirement

**Statement:** When the firmware detects the Xbox **B** button transitioning to pressed, it shall, from any state, immediately set motor PWM duty to 0 on both sides and transition the State Machine to **DISARMED**.

**Priority:** Must Have
**Status:** Draft

## Traceability

| Traces To | ID | Description |
|-----------|-----|-------------|
| Customer Need | CN.3 | Drive all motor outputs to zero on explicit disarm/e-stop command |
| Customer Problem | CP.3 | The robot must stop immediately on operator command or link loss |

## Acceptance Criteria

- [ ] With robot ARMED and motors commanded, pressing B sets measured PWM duty to 0 within one input loop cycle (≤ 10 ms)
- [ ] After B press, the State Machine reports DISARMED
- [ ] B press while in FAULT, PAIRING, or DISARMED produces no motor activation and leaves the State Machine in DISARMED
- [ ] Status LED switches to the DISARMED pattern within 1 s of B press

## Notes

The dedicated e-stop button is intentionally distinct from the arming combo (FR-005) and the fault-clear action (FR-019) so that operator intent is unambiguous in every direction.
