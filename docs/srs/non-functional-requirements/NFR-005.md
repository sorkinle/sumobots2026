# NFR-005: LED Pattern Distinguishability

## Requirement

**Category:** Usability
**Statement:** The status LED patterns for the four operator-visible modes (PAIRING, DISARMED, ARMED, FAULT) shall be visually distinguishable by a human operator at ≥ **1 m** distance under typical indoor lighting, and the LED shall reach its new pattern within **1 s** of the underlying State Machine mode change.

**Priority:** Should Have
**Status:** Draft

## Traceability

| Traces To | ID | Description |
|-----------|-----|-------------|
| Customer Need | CN.10 | Display current mode on the status LED using distinct patterns |
| Applies To FRs | FR-014 | LED patterns |
| Customer Problem | CP.7 | The operator expects clear local feedback on robot state |

## Acceptance Criteria

- [ ] Four patterns identifiable correctly by a naive observer in a 4-out-of-4 blind trial at 1 m
- [ ] Stopwatch test: mode change → new LED pattern visible within 1 s, repeated for each transition
- [ ] LED patterns remain correct at full motor load (no glitches caused by main-loop slowdown)

## Measurement Method

Blind A/B observation test with one or two observers + bench mode-switching script.
